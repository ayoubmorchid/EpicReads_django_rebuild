# EpicReads

EpicReads is a Django-based digital library project for browsing books, managing reading actions, and using simple authentication pages for readers.

## Features

- Browse a digital book catalog
- View book details
- Register and log in as a reader
- Save and manage favorite books
- Checkout and payment flow pages
- Responsive user interface

## Technologies

- Django
- Docker
- PostgreSQL
- HTML, CSS, and JavaScript

## Run With Docker

Make sure Docker is installed and running, then start the project:

```bash
docker compose up --build
```

To stop the containers:

```bash
docker compose down
```

## Local URL

Open the app in your browser:

```text
http://localhost:8000
```
